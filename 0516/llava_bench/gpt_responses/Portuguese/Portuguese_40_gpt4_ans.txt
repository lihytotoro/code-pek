A imagem é uma montagem de três painéis que juntos criam uma piada visual.

No primeiro painel, vemos um celular preto com uma capa preta sobre uma superfície de tecido bege. Um cabo branco de carregamento parece estar conectado a ele, mas, ao invés de um conector de carregamento típico, há um conector azul grande que se assemelha a um conector VGA, que é um tipo de conector geralmente usado para monitores de computador, não para carregar celulares.

No segundo painel, é mostrada uma embalagem de um produto com texto em japonês e a imagem de um cabo com o mesmo conector azul grande do primeiro painel. A embalagem indica que é um acessório para um "Lightning Cable", que é o tipo de cabo usado para carregar dispositivos da Apple. A piada aqui é que alguém poderia pensar que é um acessório real para o cabo Lightning.

No terceiro painel, é revelado o truque: uma mão está segurando o conector azul, que na verdade é uma tampa de proteção para o cabo Lightning. A tampa é colocada de tal forma que parece que o conector VGA está conectado ao celular, o que é tecnicamente impossível e absurdo, criando um efeito cômico.

A piada é um jogo de expectativas, onde inicialmente parece que alguém tentou conectar um cabo incompatível a um celular, mas na verdade é apenas uma tampa protetora para o verdadeiro conector de carregamento.
