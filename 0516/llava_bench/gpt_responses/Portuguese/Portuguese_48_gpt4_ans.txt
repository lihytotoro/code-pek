Aqui está um exemplo de código HTML e JavaScript que você pode usar para criar um site simples baseado no mock-up fornecido. Este exemplo inclui uma piada real e um botão para revelar a resposta.

```html
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>My Joke Website</title>
<style>
  body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
    text-align: center;
    padding: 50px;
  }
  #joke-container {
    background-color: #fff;
    border-radius: 8px;
    padding: 20px;
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
    margin-bottom: 20px;
  }
  #punchline {
    display: none;
    color: #007BFF;
    margin-top: 20px;
  }
  button {
    background-color: #007BFF;
    color: #fff;
    border: none;
    padding: 10px 20px;
    border-radius: 5px;
    cursor: pointer;
    font-size: 16px;
  }
  button:hover {
    background-color: #0056b3;
  }
</style>
</head>
<body>

<h1>My Joke Website</h1>

<div id="joke-container">
  <div id="joke">Why don't scientists trust atoms?</div>
  <button onclick="revealPunchline()">Push to reveal punchline</button>
  <div id="punchline">Because they make up everything!</div>
</div>

<script>
function revealPunchline() {
  document.getElementById('punchline').style.display = 'block';
}
</script>

</body>
</html>
```

Este código cria um site com um cabeçalho, uma caixa de piada, um botão para revelar a resposta e a resposta escondida. Quando o usuário clica no botão, a resposta é revelada. Os estilos são simples e você pode personalizá-los conforme necessário para adicionar mais cores ou alterar o layout.
