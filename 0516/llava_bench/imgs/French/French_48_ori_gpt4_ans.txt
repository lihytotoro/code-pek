Voici un code HTML et JavaScript simple pour créer le site web humoristique basé sur le sketch :

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mon Site Web Humoristique</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color : bleu clair;
            text-align : centre;
        }
        h1 {
            color : bleu sombre;
            margin-bottom : 50px;
        }
        p#joke {
            font-size : 24px;
            color : vert sombre;
            margin-bottom : 30px;
        }
        button {
            font-size : 18px;
            color : blanc;
            background-color : orange foncé;
            padding : 10px 20px;
            cursor : pointer;
            border : none;
        }
    </style>
</head>
<body>
    <h1>Mon Site Web Humoristique</h1>
    <p id="joke">Blague amusante</p>
    <button onclick="getJoke()">Appuyez pour révéler la punchline</button>

    <script>
        async function getJoke() {
            const response = await fetch('https://api.chucknorris.io/jokes/random');
            const data = await response.json();
            document.getElementById('joke').innerHTML = data.value;
        }
    </script>
</body>
</html>
```

Ce code créé une page web simple et colorée avec un titre "Mon Site Web Humoristique", un espace réservé pour la blague, et un bouton pour révéler la punchline. Il récupère une blague aléatoire à partir de l'API Chuck Norris Jokes (https://api.chucknorris.io/jokes/random) lorsque l'utilisateur clique sur le bouton.
